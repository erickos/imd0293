# Atividade: Hashing

Foobar is a Python library for dealing with word pluralization.

## Instalação

Use o gerenciador de pacotes [pip](https://pip.pypa.io/en/stable/) para instalar o módulo `hashlib`.

```bash
pip install hashlib
```
